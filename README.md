# load-balancer-dashboard
Load Balancer Performance Analytics Dashboard**  
An interactive dashboard built using Streamlit to analyze and compare the performance of different load balancing algorithms.  
Features

* Compare algorithms like Round Robin, Weighted Round Robin, Least Response Time, and Chained Failover  
* Visualize key metrics: **Latency, Throughput, and Error Rate**  
* Interactive controls to simulate varying number of requests and servers  
* Dynamic charts for performance comparison and trends

Tech Stack

* Python  
* Streamlit  
* Pandas  
* NumPy  
* Matplotlib

Dashboard Highlights

* Real-time simulation of load balancing scenarios  
* Comparative analysis of algorithm efficiency  
* Identification of best-performing algorithm based on metrics

Run Locally

**pip install \-r requirements.txt**

**streamlit run app.py**

Project Purpose

This project demonstrates system design concepts and data visualization skills by analyzing how different load balancing strategies impact performance in distributed systems.
